# Court-Agent
Legal intelligence agent for Ukrainian court decisions and regulatory monitoring.
- Queries EDRSR (Єдиний державний реєстр судових рішень)
- Monitors ProZorro tenders (CPV codes 79210000-1, 72000000-5, 72212700-4)
- Tracks AMCU decisions and fines

## Deployment
```bash
./deploy.sh
```
