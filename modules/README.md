# AuditorSEC Extension Modules

This directory contains AuditorSEC extension modules that expand Audityzer's capabilities across multiple security and compliance tracks.

## Modules

### Track 1: Telegram BRICS Cybersecurity (`track1-telegram-brics/`)
Telegram-based cybersecurity monitoring and alerting pipeline targeting BRICS-region threat intelligence. Includes webhook server, audit pipeline, and BRICS-specific alerting.

### Track 2: SaaS Compliance (`track2-saas-compliance/`)
SaaS compliance engine with AI-powered scanning, CRM integration, and DApp risk auditing. Provides automated compliance checks and reporting for Web3 and traditional SaaS platforms.

### Track 3: Automation R&D (`track3-automation-rd/`)
Research and development automation modules including gig contract management, IoT digital twin simulations, and workflow orchestration.

### Infra: DigitalOcean Deployment (`infra/`)
Infrastructure-as-code for deploying Audityzer on DigitalOcean. Includes Terraform configurations, Helm charts, Docker builds, CI/CD pipelines, and bootstrap scripts.
